# Eniac E-commerce Discount Strategy Analysis

This project analyzes Eniac's pricing and discount strategy using transactional sales data.

## Files
- 01_data_exploration.ipynb
- 02_data_cleaning.ipynb
- 03_feature_engineering.ipynb
- 04_discount_analysis.ipynb
- 05_seasonality_analysis.ipynb

## Datasets
- orders_cl.csv
- products_cl(1).csv
- orderlines_qu.csv
- brands(1).csv

## Main business questions
- How many products are being discounted?
- How big are the offered discounts?
- What is the distribution of prices across categories?
- How do seasonality and special dates affect sales?
- How can data collection be improved?

## Notes
The notebooks are aligned with a concise board-style storyline:
1. Understand the data
2. Clean the data
3. Build business features
4. Analyze discount behavior
5. Evaluate seasonality
